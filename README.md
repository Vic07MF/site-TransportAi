# 🚀 Transport AI

Aplicação web desenvolvida para exibir informações sobre o TransportAI.

Este projeto faz parte do Trabalho de Conclusão de Curso (TCC), com foco no uso de tecnologia para otimização de processos.

## 🌐 Acesse o projeto
https://site-transport-ai.vercel.app

## 📌 Sobre o sistema
O Transport AI é uma plataforma que permite visualizar, organizar e analisar informações relacionadas ao transporte, facilitando a tomada de decisão.

## 🛠️ Tecnologias utilizadas
- React / Next.js
- JavaScript / TypeScript
- Vercel (deploy)

## ⚙️ Funcionalidades
- Interface interativa
- Visualização de dados
- Organização de informações
